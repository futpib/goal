# goal

[![Coverage Status](https://coveralls.io/repos/github/futpib/goal/badge.svg?branch=master)](https://coveralls.io/github/futpib/goal?branch=master)

Hierarchical goal tracker CLI.

## Usage

```
goal add <description> [--parent <id>] [--continuous]
goal list
goal done <id>
goal undone <id>
goal modify <id> [--body <description>] [--parent <id>] [--no-parent] [--continuous] [--achievable]
goal delete <id>
```
